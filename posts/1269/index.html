Mac下Homebrew常用命令

发表于2022-03-01|更新于2022-06-01|技术开发

brewhome 国内镜像
/bin/zsh -c “$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)”

查看帮助信息
brew help

查看版本
brew -v

更新 brew
brew update

安装
brew install XX

查询可更新的包
brew outdated

更新包
brew upgrade 全部
brew upgrade 包名 指定包

清理旧版本
brew cleanup

# 锁定不想更新的包

锁定某个包
brew pin $FORMULA
取消锁定
brew unpin $FORMULA
查看包信息
brew info 包名

查看安装列表
brew list

查询可用包
brew search [包名]

# 服务管理

查看服务列表
brew services list
启动服务
brew services start 服务名
停止服务
brew services stop 服务名
重启服务
brew services restart 服务名
清除已卸载应用无用的配置
brew services cleanup
